## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Wrong OTLP log property key | Used `management.opentelemetry.logging.export.otlp.endpoint` which does not exist in Spring Boot 4.x. Correct key is `management.otlp.logging.export.endpoint`. Fixed in commit 6ba6aed. | advisory | Always verify Spring Boot actuator property namespaces — `management.opentelemetry.*` is for tracing only; `management.otlp.*` covers metrics AND logs |
| Distroless healthcheck tools | All observability images (otel-collector, jaeger, loki, prometheus, grafana) lack wget AND curl. Fixed: otel-collector has no healthcheck, prometheus uses promtool, loki/grafana use nc. | advisory | Check image type before writing healthchecks — otelcol-contrib is distroless, prom/* is busybox-based |

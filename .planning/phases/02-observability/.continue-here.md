## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| `docker compose` instead of `podman compose` | Rejected by user — Docker Desktop not installed. This machine uses Podman (rootful, WSL2). | blocking | Always use `podman compose` for all container operations. Convention added to CLAUDE.md. |
| `-DskipTests` instead of `-Dmaven.test.skip=true` | `-DskipTests` skips test execution but still compiles tests. ledger-service test had missing `PostgreSQLContainer` class, causing build failure mid-reactor. | blocking | Use `-Dmaven.test.skip=true` for any build where tests are not the goal. |
| JAR locked by Podman WSL2 after `podman compose down` | `mvn clean` fails with "file used by another process" even after containers are stopped. WSL2 machine holds file handles. | advisory | `podman machine stop` → delete target/ → `podman machine start` → rebuild. |
| Wrong logging property key (prior session) | `management.otlp.logging.export.endpoint` does not exist in SB 4.0.5. Correct: `management.opentelemetry.logging.export.otlp.endpoint`. Already fixed in e346ae5. | advisory | Verify management.* keys from JAR metadata before committing. |
| Smoke test `((PASS++))` under `set -e` | `((0++))` is falsy, kills script after first check. Fixed: `PASS=$((PASS + 1))`. | advisory | Use `VAR=$((VAR + 1))` pattern in bash scripts with `set -e`. |
| Distroless image healthchecks | Loki 3.7.0 has no shell/nc — CMD-SHELL healthchecks fail silently. Already fixed. | advisory | Before writing a healthcheck for any Grafana image, verify shell tools are available. |
